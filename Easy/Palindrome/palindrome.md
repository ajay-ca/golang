#Palindrome Number

#Exercise

Palindrome number is a number which reads the same when read from both sides. Find whether the given number is a palindrome or not.

#Example

**Input**
```go
Enter your number:
42524
```
**Output:**
```The given number is a palindrome```

**Input:**
```go
Enter your number:
2020
```go

**Output:**
```The given number is not a palindrome```
